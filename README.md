Pipeline Text
========================

This micro app servers as a webhook receive for New Relic to send text notifications through Twilio's API to members of your team.

Usage
-----

Need to set the following ENV vars:

* TWILIO_SID
* TWILIO_TOKEN
* TWILIO_NUMBER
